01. Filosofía PRISMA

PRISMA es un lenguaje de programación estructurado que prioriza la legibilidad. Su estructura de módulos facilita la organización de proyectos complejos, separando claramente las definiciones globales de la lógica principal.

Modularidad

Todo el código vive dentro de un MODULO. Esto previene conflictos de nombres y permite una escalabilidad limpia.

Tipado Fuerte

Desde ENTERO hasta BOOLEANO, los tipos están claramente definidos para evitar errores comunes durante la ejecución.

04. Operadores y Lógica

Las herramientas para manipular datos y comparar resultados dentro de expresiones.

Operaciones Lógicas Y, O, NO AND, OR y NOT respectivamente.

05. Galería de Ejemplos

Explora la versatilidad de PRISMA a través de estos 10 casos prácticos que cubren desde conceptos básicos hasta lógica avanzada.

01_hola_mundo.prisma
MODULO HolaMundo
    PRINCIPAL
        IMPRIME("¡Bienvenido al lenguaje PRISMA!")
    FIN-PRINCIPAL
FIN-MODULO
02_tipos_datos.prisma
MODULO TiposDatos
    VARIABLES
        ENTERO edad = 25
        DECIMAL pi = 3.1416
        TEXTO ciudad = "Ocotlán"
        BOOLEANO es_estudiante = V
    FIN-VARIABLES
    PRINCIPAL
        IMPRIME(ciudad)
    FIN-PRINCIPAL
FIN-MODULO
03_calculadora_basica.prisma
MODULO Calculadora
    VARIABLES
        ENTERO a = 15
        ENTERO b = 4
        ENTERO resultado = 0
    FIN-VARIABLES
    PRINCIPAL
        resultado = a + b // Suma
        resultado = a % b // Módulo (Residuos)
        IMPRIME(resultado)
    FIN-PRINCIPAL
FIN-MODULO
04_verificar_edad.prisma
MODULO Acceso
    PRINCIPAL
        ENTERO edad = 0
        LEER(edad)
        SI (edad >= 18) ENTONCES
            IMPRIME("Acceso Concedido")
        SINO
            IMPRIME("Acceso Denegado")
        FIN-SI
    FIN-PRINCIPAL
FIN-MODULO
05_tabla_multiplicar.prisma
MODULO Tablas
    PRINCIPAL
        ENTERO i = 0
        ENTERO tabla = 5
        PARA (i = 1 , i <= 10 , i = i + 1)
            IMPRIME(tabla * i)
        FIN-PARA
    FIN-PRINCIPAL
FIN-MODULO
06_conteo_regresivo.prisma
MODULO Cohete
    PRINCIPAL
        ENTERO cuenta = 10
        MIENTRAS (cuenta > 0)
            IMPRIME(cuenta)
            cuenta = cuenta - 1
        FIN-MIENTRAS
        IMPRIME("¡Despegue!")
    FIN-PRINCIPAL
FIN-MODULO
07_area_triangulo.prisma
MODULO Geometria
    FUNCION DECIMAL CalcularArea(DECIMAL base, DECIMAL altura)
        DECIMAL area = (base * altura) / 2.0
        RETORNA area
    FIN-FUNCION

    PRINCIPAL
        DECIMAL result = CalcularArea(10.5, 5.0)
        IMPRIME(result)
    FIN-PRINCIPAL
FIN-MODULO
08_sistema_seguridad.prisma
MODULO Seguridad
    VARIABLES
        BOOLEANO tiene_llave = V
        BOOLEANO conoce_clave = F
    FIN-VARIABLES
    PRINCIPAL
        SI (tiene_llave Y conoce_clave) ENTONCES
            IMPRIME("Puerta Abierta")
        SINO
            IMPRIME("Alarma Activada")
        FIN-SI
    FIN-PRINCIPAL
FIN-MODULO
09_mensajes_dinamicos.prisma
MODULO Saludos
    PRINCIPAL
        TEXTO usuario = "Admin"
        IMPRIME("Sesión iniciada como: " + usuario)
    FIN-PRINCIPAL
FIN-MODULO
10_par_impar_detector.prisma
MODULO Detector
    PRINCIPAL
        ENTERO num = 0
        IMPRIME("Introduce un número:")
        LEER(num)
        
        SI (num % 2 == 0) ENTONCES
            IMPRIME("Es un número PAR")
        SINO
            IMPRIME("Es un número IMPAR")
        FIN-SI
    FIN-PRINCIPAL
FIN-MODULO
